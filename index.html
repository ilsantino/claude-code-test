Widget Status

The floating chat button should be visible in the bottom-right corner. Click it to open the chat panel.

Features to Test

• Click the bubble button to toggle the chat window
• Type a message and press Enter or click Send
• Click the mic button to use voice input (Chrome/Edge recommended)
• The widget sends messages to the configured n8n webhook
• Session ID persists in sessionStorage until tab closes
